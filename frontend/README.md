# MatterGen UI (Vite + React)

## Requirements
- Node.js 18+

## Configure API base (optional)
If the API is not on the same origin, create `frontend/.env.local`:

VITE_API_BASE=http://<server-ip>:8000

## Run dev server

npm install
npm run dev

The dev server uses `--host`, so it is reachable from other devices.

## Build

npm run build
npm run preview
